##### راهنمای استفاده از مستندات
این مستند پر است از دستورات و نمونه کانفیگ هایی که در تجربه‌های مختلف در اینجا مستند شده‌اند. تمام دستورات و تنظیمات در داخل Code Block قرار گرفته اند. سعی کردم که استانداردی برای مطالعه بهتر داشته باشم. اولین موضوع اینکه اگر قبل از دستور $ بود، یعنی می توان با کاربر عادی دستور را اجرا کرد و اگر قبل از دستور # بود یعنی باید با کاربر root آنرا اجرا کنید.\\
در تنظیمات و دستورات بخش‌هایی که در داخل < > قرار دارند یعنی شما باید با مقدار خودتون جایگزین کنید و فیلد مورد نظر اجباری است. و اگر بخشی از دستور یا پیکربندی بین [ ] قرار داشته باشد، فیلد مورد نظر اختیاری است.\\
