## Copy-mode
Copy text between panes.  

| Command | Description |
| --- | --- |
| `Ctrl`+`b`,`[` | Enter copy-mode   |
| `Ctrl`+`space` | Start selecting   |
| `Alt`+`w` or `Ctrl`+`w`  | Copy to clipboard | 
| `Ctrl`+`b`,`]` | Paste copied text from tmux clipboard. |
